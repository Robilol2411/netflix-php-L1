CINEMAX

Bienvenue sur CINEMAX

Revendeur officiel des meilleurs films du moment. Découvrez, achetez et gérez votre collection de films en ligne.

⚠️ Note : Ce site est une application PHP dynamique avec base de données MySQL. Pour l'utiliser pleinement, installez-le en local en suivant les instructions ci-dessous. GitHub Pages ne prend pas en charge l'exécution de PHP.
Voir le code source Documentation
🎬

Catalogue de films

Parcourez notre large sélection de films classés par catégories : action, horreur, animation, documentaire et drama.

🔍

Recherche avancée

Trouvez rapidement un film grâce à la barre de recherche connectée à l'API TMDB.

🛒

Panier d'achat

Ajoutez vos films préférés à votre panier et finalisez votre commande en quelques clics.

👤

Profil utilisateur

Créez votre compte, connectez-vous et gérez vos films achetés depuis votre profil personnel.

Installation en local

1

Installer Docker

Téléchargez et installez Docker Desktop, puis lancez-le.

2

Cloner le projet

Dans un terminal : git clone https://github.com/Robilol2411/netflix-php-L1

3

Configurer les fichiers d'environnement

Dans le dossier baseDD/, créez env.php et envapi.php à partir des fichiers exemples fournis.

4

Importer la base de données

Ouvrez http://localhost:8080, créez une base de données et importez le fichier monsite.sql.

5

Lancer l'application

Ouvrez http://localhost dans votre navigateur. Le site est prêt !
